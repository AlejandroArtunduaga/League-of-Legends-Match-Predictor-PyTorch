# League of Legends Match Predictor — PyTorch

Modelo de regresión logística implementado 
desde cero con PyTorch para predecir 
victorias en League of Legends.

## Pipeline completo
- Preprocesamiento y estandarización
- Entrenamiento con backpropagation manual
- Regularización L2 (Ridge)
- Evaluación: Accuracy, ROC-AUC, 
  Matriz de Confusión
- Hyperparameter tuning (learning rates)
- Feature importance analysis
